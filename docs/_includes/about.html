What is MinVar?

A command-line tool to discover drug resistance mutations in HIV-1 populations using deep sequencing data. Developed at IMV, UniZurich.

Designed for clinicians

Only one input file in FASTQ format is required, no need to set parameters. The final output is a PDF report with annotated DRM (annotations according to HIVDB).

Accurate

On an extensive set of control experiments, MinVar reliably called variants down to 5%. In the paper we recommend and motivate this as an accurate detection threshold.

Quick

Run time is negligible compared to the overall experiment.

Versatile

MinVar was successfully tested on reads from both Roche/454 and Illumina MiSeq sequencing platforms.
